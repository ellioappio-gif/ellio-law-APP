# ellio-law Logo Asset

The ellio elephant logo should be placed in this directory.

## File requirements:
- **Format**: PNG with transparency, or SVG
- **Filename**: `ellio-logo.png` or `ellio-logo.svg`
- **Sizes needed**:
  - 1x: 120x120px
  - 2x: 240x240px
  - 3x: 360x360px
  - SVG: scalable vector

## Brand guidelines:
- Purple/blue gradient elephant
- Lowercase "ellio" wordmark
- Never modify colors
- Never add effects
- Maintain aspect ratio

## Usage in app:
```tsx
import EllioLogo from '../assets/images/ellio-logo.png';

<Image source={EllioLogo} style={{ width: 60, height: 60 }} />
```
